## 🚀 Quick Start

### Installation

```bash
# Install from source
pip install -e .

# Set API keys
export DEEPSEEK_API_KEY="sk-..."  # Required (best value)
export XAI_API_KEY="xai-..."      # For Grok models
```

### First Project

```bash
python -m orchestrator \
  --project "Build a FastAPI authentication service" \
  --criteria "All endpoints tested, OpenAPI docs complete" \
  --budget 5.0
```

### Run Tests

```bash
# Run all tests
pytest tests/ -v

# Run with coverage
pytest tests/ --cov=orchestrator --cov-report=html
```
